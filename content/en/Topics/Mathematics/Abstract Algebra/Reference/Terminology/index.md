This glossary gathers the most important definitions from the public abstract algebra track. Longer entries live on their own subpages so the list can stay readable as it grows.

## Current Entries

- [Binary Operation](./Binary%20Operation.md): a rule combining two elements of a set and staying inside the same set.
- [Identity](./Identity.md): an element that leaves every element unchanged under the operation.
- [Inverse](./Inverse.md): an element that combines with a given element to produce the identity.
- [Group](./Group.md): a set with a binary operation satisfying closure, associativity, identity, and inverse axioms.
- [Subgroup](./Subgroup.md): a subset that is itself a group under the inherited operation.
- [Cyclic Group](./Cyclic%20Group.md): a group generated by one element.
- [Generator](./Generator.md): an element or set of elements from which the whole group can be built.
- [Permutation](./Permutation.md): a bijection from a set to itself.
- [Coset](./Coset.md): a translated copy of a subgroup.
- [Normal Subgroup](./Normal%20Subgroup.md): a subgroup whose left and right cosets coincide.
- [Homomorphism](./Homomorphism.md): a structure-preserving map between groups.
- [Kernel](./Kernel.md): the set of elements sent to the identity by a homomorphism.
- [Factor Group](./Factor%20Group.md): a quotient group built from cosets of a normal subgroup.
- [Simple Group](./Simple%20Group.md): a group with no nontrivial normal subgroups.
